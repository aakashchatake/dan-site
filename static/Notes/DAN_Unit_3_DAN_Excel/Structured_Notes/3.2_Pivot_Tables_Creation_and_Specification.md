# 3.2 Pivot Tables: Creation and Specification

## Creating Pivot Tables\n- Select your dataset and choose Insert â†’ PivotTable.\n- Decide whether to place the pivot table in a new worksheet or existing one.\n- If using an Excel table, the pivot table will expand automatically when new rows are added.\n\n## Specifying Fields\n- Drag fields to the **Rows** area to group by categories (e.g., region, product).\n- Drag numerical fields to the **Values** area to summarise (sum, count, average).\n- Use the **Columns** area to create cross-tabulations across categories.\n- The **Filters** area adds drop-down filters to the entire pivot table.\n\n## Examples\n- Summarising revenue by quarter and region by placing Region in Rows, Quarter in Columns and Revenue in Values.\n- Filtering a pivot table to display data for a specific salesperson.\n\n## Summary\nPivot tables quickly summarise large datasets. Properly choosing rows, columns, values and filters determines the insight you gain from the table.\n
